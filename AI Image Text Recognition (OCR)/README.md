# AI Image Text Recognition

An AI-powered Optical Character Recognition (OCR) application that extracts text from images using EasyOCR. The project provides both a Command Line Interface (CLI) and a Streamlit-based web application.

---

## Features

- Extract text from images
- AI-powered OCR using EasyOCR
- Supports JPG, JPEG and PNG
- Confidence score for detected text
- Download extracted text
- Streamlit Web Interface
- Command Line Interface
- Clean and Modular Code
- Unit Testing

---


