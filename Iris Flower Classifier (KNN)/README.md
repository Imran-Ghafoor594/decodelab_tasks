# Iris Flower Classification Using K-Nearest Neighbors (KNN)

## Project Information
This project was developed as part of the DecodeLabs Artificial Intelligence Internship Project 2.

## Objective
Build a machine learning classification model using the Iris dataset and classify flower species using the K-Nearest Neighbors (KNN) algorithm.

## Dataset
The Iris dataset contains:
- 150 samples
- 4 input features
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- 3 classes
  - Iris Setosa
  - Iris Versicolor
  - Iris Virginica

## Technologies Used
- Python

- Pandas
- Scikit-Learn
- Jupyter Notebook
