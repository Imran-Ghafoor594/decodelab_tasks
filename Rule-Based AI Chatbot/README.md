# Rule-Based AI Chatbot

## Overview
A simple rule-based chatbot developed using Python.

## Features

- Continuous conversation loop
- Input sanitization
- Knowledge-base dictionary
- 5+ predefined intents
- Default fallback response
- Exit strategy

## Technologies

- Python

## Project Logic

User Input
↓
Sanitize Input
↓
Check Exit Command
↓
Dictionary Lookup
↓
Response / Fallback

## Run

python chatbot.py

## Example

You: hi
Bot: Hello! How can I help you?

You: help
Bot: I can answer simple predefined questions.

You: bye
Bot: Goodbye!
